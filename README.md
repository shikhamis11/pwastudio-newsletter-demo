## Installation

`git init`

`git remote add origin https://github.com/shikhamis11/pwastudio-newsletter-demo.git`

`git pull origin master`

run below commands

`yarn install`

`yarn buildpack create-env-file ./`

update .env file for backend URL and other things then run

`yarn watch`
